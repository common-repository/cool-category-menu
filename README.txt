== Description ==

Cool Category Menu

This plugin better stylizes the category menu that comes with the WordPress Flash theme.
It allows you to change the color of fonts, borders, selected elements and spacing between items.
In addition, it has better animation for opening and closing menus.

== Installation ==

e.g.

1. Load the plug-in files into the directory `/ wp-content / plugins / plugin-name` or install the plug-in directly from the WordPress plugin page.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Cool Category Menu screen to configure the plugin

== Frequently Asked Questions ==

= How to change the font color of the selected element? =

On the plugin page select the desired color.
